Markdown插入数学公式

Posted on 2019-12-29 | |

  在markdown里面插入数学公式, 首先需要选取MathJax引擎, 把以下代码插入md文件即可.

  如果使用hexo之类的框架可以手动配置Mathjax引擎,这样就不需要每次都插入这一行代码.

行内插入

  这是一行(a + b)
